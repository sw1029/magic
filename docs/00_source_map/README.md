# 원본 논의 맵

이 디렉토리는 `chat/`에 남아 있는 원본 논의를 구현 관점에서 다시 읽기 쉽게 정리한 영역입니다.

문서 구성:

* [`discussion-timeline.md`](discussion-timeline.md): `request-answer01`부터 `18`까지의 순서별 흐름
* [`idea-lineage.md`](idea-lineage.md): 논의를 4단계로 압축한 계보 문서
* [`client-decisions.md`](client-decisions.md): 현재 기준으로 고정된 클라이언트 결정

읽는 순서:

1. [`client-decisions.md`](client-decisions.md)
2. [`idea-lineage.md`](idea-lineage.md)
3. [`discussion-timeline.md`](discussion-timeline.md)

이 영역의 역할은 “왜 이런 task가 생겼는가”를 설명하는 것입니다. 실제 구현 상세는 [`../30_tasks/README.md`](../30_tasks/README.md) 이하를 봅니다.
