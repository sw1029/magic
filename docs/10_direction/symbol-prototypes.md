# 기본 심볼 5종 임시 시안

이 문서는 현재 프로토타입 기준으로 사용하는 기본 심볼 5종의 임시 방향을 기록한 문서입니다.

이 시안은 최종 아트가 아닙니다.
현재 목적은 아래 두 가지입니다.

* 입력이 쉬운가
* 서로 헷갈리지 않고 읽히는가

---

## 공통 원칙

* 최대한 단순할 것
* 획 수가 적을 것
* 빠르게 그릴 수 있을 것
* 서로 모양이 충분히 다를 것
* 프로토타입 단계에서는 미적 완성도보다 판독성을 우선할 것

---

## 현재 임시 시안

### 바람

* 현재 시안: 가로 3획
* 의도: 빠른 흐름과 가벼운 방향성을 직관적으로 보이게 함
* 입력/판독 관점 메모: 다른 심볼보다 선형적이고 가벼운 인상이 분명해야 함

### 땅

* 현재 시안: 사다리꼴
* 의도: 아래가 넓고 안정적인 구조를 바로 읽히게 함
* 입력/판독 관점 메모: 무게감과 바닥에 닿는 인상이 중요함

### 불꽃

* 현재 시안: 삼각형
* 의도: 위로 치솟는 인상을 가장 단순하게 표현
* 입력/판독 관점 메모: 땅과 혼동되지 않게 방향성이 살아 있어야 함

### 물

* 현재 시안: 동심원 또는 물방울
* 의도: 흐름, 파문, 반복감을 보이게 함
* 입력/판독 관점 메모: 바람과 헷갈리지 않게 닫힘 또는 파문 인상이 필요함

### 생명

* 현재 시안: 새싹 모양
* 의도: 자람, 발아, 분기를 직관적으로 보이게 함
* 입력/판독 관점 메모: 물과 달리 생장 방향성이 드러나야 함

---

## 현재 시안의 용도

이 문서는 아래 task들의 기준 시안으로 사용한다.

* 기본 문양 입력 규칙 정리
* 기본 문양 판독 기준 정리
* 속도와 각도에 따른 결과 차이 테스트
* 허수아비 대상 프로토타입 검증

즉, 지금 단계에서는 “멋진 최종 문양”보다
**입력과 결과 시스템을 먼저 검증하기 위한 기준형**이다.

---

## 이후 보강 예정 요소

현재는 아래를 후속 작업으로 둔다.

* 최종 심볼 디자인 보강
* 학파별 연출 차이
* 더 정교한 형태 구분
* 시각 완성도 조정
