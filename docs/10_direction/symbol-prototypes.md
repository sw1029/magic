# 기본 심볼 5종 임시 시안

이 문서는 현재 프로토타입 기준으로 사용하는 기본 심볼 5종의 임시 방향을 기록한 문서입니다.

이 시안은 최종 아트가 아닙니다.
현재 목적은 아래 두 가지입니다.

* 입력이 쉬운가
* 서로 헷갈리지 않고 읽히는가

---

## 공통 원칙

* 최대한 단순할 것
* 획 수가 적을 것
* 빠르게 그릴 수 있을 것
* 서로 모양이 충분히 다를 것
* 프로토타입 단계에서는 미적 완성도보다 판독성을 우선할 것

---

## 현재 임시 시안

### 바람

* 현재 시안: 3개의 평행 개방선
* 의도: 빠른 흐름과 가벼운 방향성을 직관적으로 보이게 함
* 입력/판독 관점 메모: 폐합하지 않고, 세 줄이 거의 평행하며, 간격이 완전히 무너지지 않아야 함

### 땅

* 현재 시안: 하변이 더 긴 폐합 사다리꼴
* 의도: 아래가 넓고 안정적인 구조를 바로 읽히게 함
* 입력/판독 관점 메모: 닫힌 다각형이어야 하며, 삼각형처럼 위로 과도하게 모이면 안 됨

### 불꽃

* 현재 시안: 상향 인상의 폐합 삼각형
* 의도: 위로 치솟는 인상을 가장 단순하게 표현
* 입력/판독 관점 메모: 닫힌 3각 구조가 핵심이며, 땅과 달리 네 번째 지지면을 두지 않음

### 물

* 현재 시안: 단일 원형 폐합 루프
* 의도: 흐름, 파문, 반복감을 보이게 함
* 입력/판독 관점 메모: V1 recognizer 기준형은 원형 하나로 고정하고, 물방울/동심원 variant는 후속 단계로 미룸

### 생명

* 현재 시안: 줄기와 상단 분기가 있는 rooted Y
* 의도: 자람, 발아, 분기를 직관적으로 보이게 함
* 입력/판독 관점 메모: 폐합하지 않고, 가지치기 구조가 살아 있어야 하며, 복잡한 새싹 변형은 V1에서 허용하지 않음

---

## 현재 시안의 용도

이 문서는 아래 task들의 기준 시안으로 사용한다.

* 기본 문양 입력 규칙 정리
* 기본 문양 판독 기준 정리
* 속도와 각도에 따른 결과 차이 테스트
* 허수아비 대상 프로토타입 검증

즉, 지금 단계에서는 “멋진 최종 문양”보다
**입력과 결과 시스템을 먼저 검증하기 위한 기준형**이다.

V1 recognizer에서는 family마다 위 기준형 하나만 canonical silhouette로 사용한다.
