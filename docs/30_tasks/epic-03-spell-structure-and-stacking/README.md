# Epic 03. spell-structure-and-stacking

## 요약

한 마법진 안의 구조, 성질 변형 여러 겹 허용 규칙, 여러 마법진 연결 규칙, 無와 武의 역할 차이를 정리하는 epic이다.

## 아이디어 원본

초기 분류 체계, 마법진 작성 양식, 단순화된 기본 심볼 구조, 클라이언트의 다중 성질 변형 허용 의견을 함께 반영한다.

## 구현하고자 하는 방향

단일 구조와 복합 구조를 모두 문서상 결정 완료 상태로 만들고, 현재 기준에 맞는 중첩과 연결 규칙을 정리한다.

## 구현 의도

이 epic의 목적은 “어디까지 한 마법진 안에서 해결하고, 어디서부터 여러 마법진 연결로 넘길 것인가”를 명확히 하는 것이다.
