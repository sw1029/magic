# `docs/` 문서 허브

`chat/`는 원본 논의 보관소로 유지하고, `docs/`는 실제 구현을 위한 정리 문서 체계로 사용합니다.

이 디렉토리는 아래 네 축으로만 구성됩니다.

* [`00_source_map/README.md`](00_source_map/README.md): 원본 논의 흐름과 결정 근거 정리
* [`10_direction/README.md`](10_direction/README.md): 현재 구현 방향과 프로토타입 목표 정리
* [`20_queue/README.md`](20_queue/README.md): 작업 큐, 단계별 로드맵, 문서 규칙
* [`30_tasks/README.md`](30_tasks/README.md): epic 및 상/하위 task 문서

빠르게 시작하려면 아래 허브 문서를 순서대로 읽습니다.

1. [`00_source_map/README.md`](00_source_map/README.md)
2. [`10_direction/README.md`](10_direction/README.md)
3. [`20_queue/README.md`](20_queue/README.md)
4. [`30_tasks/README.md`](30_tasks/README.md)

원본 논의를 추적해야 할 때는 먼저 [`00_source_map/README.md`](00_source_map/README.md)로 들어간 뒤, 그 안의 타임라인 문서를 기준으로 `chat/request-answer01.md`부터 `18.md`까지 거슬러 올라갑니다.
